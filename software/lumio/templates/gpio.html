🤖 LUMIO V2 Control GPIO Config
RP2040 GPIO Configuration
Assign sensors to the co-processor's free GPIO pins. Changes are sent to the RP2040 live over UART.
The RP2040 co-processor provides extra GPIO capacity beyond what the Raspberry Pi 5 uses directly. Use this panel to tell the RP2040 firmware which pins to treat as inputs or outputs, give them labels, and read their values in real-time.
Note: The RP2040 firmware must implement the gpio_write / gpio_read JSON UART commands.
